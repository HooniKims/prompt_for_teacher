## Summary

`AI 이전에 사람이 있다: 생각의 힘, 사고의 단단함` 방향을 바탕으로, 교사의 수업 준비와 행정업무를 지원하는 **질문형 프롬프트 가이드**를 구현한다.

이 기능은 복사형 프롬프트 모음집이 아니라, 교사가 산출물, 사용 장면, 교육 및 업무 맥락, 목표, 교사 판단 지점, 필요한 자료, 안전 확인, 외부 도구 및 서비스 여부, 출력 형식, 품질 기준을 차례로 점검하며 최종 프롬프트를 설계하도록 돕는다.

## Scope

- 자연어 초기 요청 입력
- 수업 준비, 행정업무, 혼합 업무 맥락 추론
- 10단계 질문형 설계 흐름
- 교사 주도성을 지키는 교사 판단 지점 확인
- 개인정보 및 학습지원 소프트웨어 관련 안전 확인
- 규정의 쉬운 말 참고 안내
- 범용 최종 프롬프트 생성
- 최종 프롬프트와 참고 안내 분리
- 복사하기 동작
- 자연어 수정 요청 기반 프롬프트 재생성

## Acceptance Criteria

- 교사는 업무 유형을 먼저 고르지 않고 자연어로 요청을 시작할 수 있다.
- 시스템은 한 번에 하나의 핵심 질문을 던진다.
- 시스템은 수업 준비와 행정업무가 섞인 요청을 처리할 수 있다.
- 시스템은 실제 학생 개인정보나 외부 서비스 사용 가능성이 보이면 쉬운 말로 안전 확인을 한다.
- 시스템은 관련 규정이 필요한 경우 `참고 안내`에 관련 기준의 해당 부분, 쉬운 설명, 현재 요청과의 연결 이유를 표시한다.
- 시스템은 `최종 프롬프트`와 `참고 안내`를 분리해서 제공한다.
- 교사는 최종 프롬프트를 별도로 복사할 수 있다.
- 교사는 참고 안내를 별도로 복사할 수 있다.
- 교사는 마음에 들지 않는 부분을 자연어로 지적할 수 있다.
- 시스템은 수정 요청을 반영해 전체 프롬프트를 다시 제공하고 변경점을 짧게 설명한다.
